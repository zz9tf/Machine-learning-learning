# SMILE Lab Experiments in 2025 Spring

## Task

Classify the file `data/test_graph_labels.txt`.  And submit a file with two columns (graph id, graph label)

### 10 lines Example

```
0,1
3,2
4,3
5,4
6,5
8,6
9,5
10,3
11,1
12,2
```

## Dataset File Description

This folder contains the following comma separated text files:

- data/A.txt  # adjacency matrix of the graph (node_id i, node_id j)
- data/*_graph_indicator.txt  # (node_id, graph_id, node_label)
- data/*_graph_labels.txt  # (graph_id, graph_label)
